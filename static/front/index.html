EmailEngine

EmailEngine is a self-hosted headless email client for accessing IMAP accounts over REST API. EmailEngine connects to the user's IMAP account, translates API requests to IMAP, and sends webhooks for changes like new or deleted emails.

To continue setting up your EmailEngine installation, log in to your server using SSH:
ssh root@example.com
